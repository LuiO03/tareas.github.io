Sobre Mí

Hola, soy Luis Quispe Osorio, un desarrollador web apasionado por crear soluciones innovadoras y eficientes. Con experiencia en tecnologías como HTML, CSS, JavaScript y frameworks modernos, me dedico a transformar ideas en realidades digitales.

Especialidades

Desarrollo Frontend

Especializado en la creación de interfaces de usuario interactivas y responsivas utilizando HTML, CSS, JavaScript y frameworks como React.

Desarrollo Backend

Diseño UI/UX
